
#  A Simple Portfolio Website

## Description
This project is a simple, responsive webpage designed to demonstrate HTML and CSS skills.  
It includes a header with navigation, a main content area, and a footer.  
The webpage features hover effects, a sticky header, and responsive layout techniques.

---
### 1. Set Up the Project
- Created a new folder for the project.
- Added `index.html` for the HTML structure.
- Added `style.css` for styling the page.
- Linked `style.css` to `index.html` using the `<link>` tag.

---
